🧠✍️ IA Redação ENEM

Uma aplicação de Inteligência Artificial para gerar redações no modelo do ENEM, ajudando estudantes a treinar escrita, estrutura e argumentação.

📚 Sobre o Projeto

O IA Redação ENEM é uma ferramenta que utiliza Inteligência Artificial para gerar redações estruturadas no padrão do ENEM, com:

Introdução

Desenvolvimento

Conclusão com proposta de intervenção

O objetivo é ajudar estudantes a praticar redação, entender a estrutura exigida pela prova e melhorar sua argumentação.

🚀 Funcionalidades

✅ Gerar redações completas a partir de um tema
✅ Estrutura padrão ENEM (Introdução + 2 desenvolvimentos + Conclusão)
✅ Sugestão de argumentos
✅ Treino para estudantes
✅ Interface simples e rápida
.

🎯 Objetivo

Este projeto foi criado para:

ajudar estudantes a treinar redação

demonstrar o uso de IA em aplicações educacionais

servir como projeto de portfólio para desenvolvedores
